===============================
Generating a QM complete model
===============================

.. contents::

Overview
--------

A program to complete a model ready for Q|R refinement. This includes hydrogens,
terminal oxygens and side-chains.

Running the program
-------------------

``qr.finalise`` can be executed on a PDB model file.
